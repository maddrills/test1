VisionPlus Navigation — Merged Component

This navbar uses every colour token from visionplus-dispensing.html exactly — the same four themes, the same CSS variable names, the same 3-D button technique from the dispensing module tabs. Drop this <header> block into your _Layout.cshtml and it will be visually consistent with the dispensing page on every theme automatically.

✓ Exact dispensing colour tokens
✓ 3-D buttons from .mt style
✓ Gold active tab (themes-aware)
✓ Quick-access below nav bar
✓ Dispensing theme picker popup
✓ Chrome · Edge · Safari · Firefox
✓ Mobile · Tablet responsive

Integration — _Layout.cshtml

The CSS tokens are already identical to visionplus-dispensing.html, so you can share a single site.css. The active page state is set with the active class on the relevant .vp-nav-item:

@{ var ctrl = ViewContext.RouteData.Values["controller"]?.ToString(); }
<div class="vp-nav-item @(ctrl=="Dispensing" ? "active" : "")" id="nav-dispensing">

Quick-access routes:

href="@Url.Action("New","Tasks")"
href="@Url.Action("Index","OptomDiary")"
